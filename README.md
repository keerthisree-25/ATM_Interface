# ATM Console Application (Java)

## Description
A console-based ATM simulation developed using Java.  
The application allows users to register, log in, and perform basic banking operations.

## Features
- User Registration & Login
- Balance Inquiry
- Deposit Money
- Withdraw Money
- Fund Transfer
- Transaction History
- Input Validation & Exception Handling

## Technologies Used
- Java
- OOP Concepts
- Console I/O
